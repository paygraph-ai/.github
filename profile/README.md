PayGraph is an open-source Python SDK that lets AI agents spend money safely. Policy engine, virtual cards, full audit trail — enforced in code, outside the LLM context.

Today we route through Stripe Issuing, Stripe MPP, and Coinbase x402. The architecture is built so your policy engine stays the same regardless of what payment rail sits underneath.
